小菜鸟进入了紧张的秋招中，对网络的知识进行了预习和复习，今天主要说说TCP的连接和断开过程。

首先大家都知道TCP和UDP这两个东西，那么他们俩一个最大的区别就是TCP是面向连接，而UDP是面向非连接。
面向连接：在正式通信前必须要和对方建立连接，比如你给别人打电话，必须要对方接了你的电话，你们俩才能够通讯，假如他拒绝了你的电话，那你也就和他通讯不了了。
面向非连接：在正式通信前不用和对方建立连接，比如你发短信，你只需要知道对方的电话号码，然后你发送过去，你管他有没有收到，反正我已经发出去了。

AsyncTask简介

AsyncTask 是一个轻量级别的异步任务类，它可以在线程池中执行后台任务，然后把结果传递给主线程中更行UI，通过AsyncTask可以更加方便的执行后台任务以及访问在主线程中访问UI。但是AsyncTask不适合耗时太长的任务，以及大量的任务。AsyncTask的底层实现主要是线程池和Handler。

理解Window的机制

发表于 2017-07-27 | 分类 学习

Window和WindowManager

Window是什么？

Window表示一个窗口的概念，Window是一个抽象类，它的唯一具体实现类是PhoneWindow，这一点可以从源码可知。

The only existing implementation of this abstract class is android.view.PhoneWindow, which you should instantiate when needing a Window.

Android中的所有视图都是附加在Window上来展示的，无论是我们非常熟悉的Activity，Toast，状态栏等等，还是很少我们很少用到的PopupWindow，他们的视图都是附加在Window上的。所以，由此来说，Window还是View的直接管理者，从View的事件分发机制就可以知道，事件首先从Activity传到Window，然后在传到DecorView，接着进行下一步的传递。
